CLI Quick Reference — Configure experiments from your terminal

firebase remoteconfig:get -o experiments/rc-snapshot.json Download current Remote Config (all parameters + values) to a local file
firebase remoteconfig:set experiments/rc-active.json Push a new config template — use to roll out a winning variant to 100% of users
firebase remoteconfig:versions:list List all published versions — find a version number to roll back to
firebase remoteconfig:rollback --version-number 5 Instantly revert Remote Config to a previous version
firebase open remoteconfig Open Firebase Remote Config console in browser — create/manage experiments here
