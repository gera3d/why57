## Publishing standard

A quantified outcome is publishable only when 57 retains all of the following:

1. a dated source file or stable source URL;
2. a written definition of the metric and comparison window;
3. enough context to explain 57's contribution without claiming sole attribution where other factors were involved; and
4. client approval to publish the client name, metric, and testimonial.

Implementation scope may be described without a performance number when a portfolio artifact supports that the work existed, but the client, production status, and scope still require owner confirmation. A testimonial needs an original capture and permission record; a quote copied from earlier site copy is not a source.

Status vocabulary:

- **Removed — unverified:** no repository evidence supports public use.
- **Withheld — source required:** the implementation may be real, but the quantified result is not publishable yet.
- **Partial — portfolio artifact only:** an image or existing portfolio record supports product scope, not performance.
- **Pending — source capture:** qualitative testimonial remains attributed but needs an original and permission record.
- **Owner attestation required:** operational business information must be confirmed by the owner.

## Evidence file naming

Store source records outside the public web root using a consistent private convention:

`YYYY-MM-DD_client_claim_metric_source.ext`

For each approved claim, record the private source location, metric definition, comparison window, calculation, approver, and approval date in this ledger without committing confidential client material to the public repository.

## Pre-publication checklist

- Re-run the public claim scan for conflicting totals, multipliers, percentages, superlatives, and guarantees.
- Confirm every visible quote has a retained original and permission record.
- Confirm structured data does not repeat removed claims.
- Confirm Remote Config cannot reintroduce stale proof copy.
- Recheck public business identity details with the owner.
- Review every case study with the client before restoring a quantified result.
